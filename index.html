Versionamento

O git é um software que permite realizae versôes de um projeto, onde o GIT-HUB desktop é um software que faz a ligação entre VSCode, GIT e GIT-HUB.

O GIT_HUB desktop permite fazer a sincronia entre o repositório local e o repositório remoto

Commit

É um processo de confirmação de uma versão entre vírgulas, ou seja, a cada alteração é possivel confirmar a alteração

Push

Esse comando envia (sincroniza) as alteraçôes realizadas no repositório local com o repositório remoto no GIT-HUB
